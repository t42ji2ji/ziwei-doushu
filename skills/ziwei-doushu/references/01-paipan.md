## §1.1 時辰原則（採本派）

**規則**：採出生**當地時間**，不需經緯度換算，也不換算為天文時。

> 「目前的紫微斗數主流看法，命盤上的時辰只需要以出生當地時間就可以，不需要換算，也就是說，如果我在日本出生，我就用日本當地時間，不需要再換算時間」

**唯一例外**：橫跨多時區但統一使用一個官方時間的地區（如中國大陸全境用北京時間、馬來西亞），需換算到實際當地時間。

**兩小時一辰**（傳統十二時辰）：

| 時辰 | 區間 | 時辰 | 區間 |
|------|------|------|------|
| 子 | 23:00–01:00 | 午 | 11:00–13:00 |
| 丑 | 01:00–03:00 | 未 | 13:00–15:00 |
| 寅 | 03:00–05:00 | 申 | 15:00–17:00 |
| 卯 | 05:00–07:00 | 酉 | 17:00–19:00 |
| 辰 | 07:00–09:00 | 戌 | 19:00–21:00 |
| 巳 | 09:00–11:00 | 亥 | 21:00–23:00 |

**夏令時**：四本書未明示。實作上需向用戶確認其出生時刻是否已校正夏令時。

**出生地未知**：採目前居住地當地時間。

**確切時辰未知**：提供「不知時辰」模式，只解年月日無法定身宮與宮位，星曜推算範圍。

## §1.2 子時規則（紫微傳統採子正派 — 已固定為預設）

**相關典籍未明確處理 23:00 的日界**。但**紫微斗數的傳統慣例與八字不同**，採子正派。

### 紫微傳統規則（預設）

> 在紫微斗數中,通常不採用八字的「早子/晚子」分法。
> **晚上 23:00 至凌晨 01:00 出生者一律歸為「子時」**。
> 因古時以子時為一日之始,過 23:00 便直接排入「次日」。

→ **23:00-23:59 算次日子時、00:00-00:59 算當日子時**。
→ Script 預設 `--zipai late`,不需要再問使用者。

### 兩派對照

| 派別 | 規則 | 用在哪 |
|------|------|--------|
| **子正派(紫微傳統,預設)** | 23:00-23:59 算次日、00:00-00:59 算當日 | 紫微斗數;本 skill 預設 |
| 早子派(八字慣例) | 23:00-00:59 都算當日 | 部分八字流派、少數現代軟體 |

### 切換

| 情境 | 動作 |
|------|------|
| 一般使用 | 不用做任何事,Script 預設子正派 |
| 使用者明確說「我要用八字早子規則」 | 加 `--zipai early` |

**不再問使用者**。除非使用者主動指明八字派,否則直接套用紫微傳統。

### 邊界 case

- 22:59 出生 → 亥時,當日排盤(沒有歸屬爭議)
- 23:00 整 出生 → 子時,**算次日**(子正派)
- 00:00 整 出生 → 子時,**算當日**(子正派)
- 00:30 出生 → 子時,當日排盤(沒有歸屬爭議)

## §1.3 大限起運

**起運歲數 = 五行局數**：

| 五行局 | 起運歲數 |
|--------|---------|
| 水二局 | 2 歲 |
| 木三局 | 3 歲 |
| 金四局 | 4 歲 |
| 土五局 | 5 歲 |
| 火六局 | 6 歲 |

**大限行運方向**：
- 陽男陰女 → 順走（順時針）
- 陰男陽女 → 逆行（逆時針）

每個大限 = 10 年。

## §1.4 流年命宮

**規則**：流年命宮 = 當年地支對應的宮位。

每年每個人的流年命宮都會落在同一個地支位置，例：
- 2018 戊戌年 → 所有人的流年命宮都在「戌」位
- 2026 丙午年 → 所有人的流年命宮都在「午」位

**等於生肖地支**。

## §1.8 斗君定義

**斗君** = 本命命盤十二地支「寅」的位置。

流年斗君 = 該流年的「一月」,依順時鐘排出十二個月。

**流月、流日的天干使用農民曆上的天干**（跟大家一樣的時間）,不另起干。

---

## §1.9 排盤層的核心心法呼應

- **核心觀念:「本命盤看天生,運限盤看現象」** → 排盤要分本命 / 大限 / 流年三層次,每層獨立排,要看時再疊宮。
- 本命盤夫妻宮 ≠ 另一半長相,只 = 你對感情的態度。這項心法從排盤層就要建立,不要把「命盤的宮 = 某個人」搞混。

## §1.10 流月流日排法與挑時機

**用途**：推算短時段運勢,主動挑時機趨吉避凶 — 談判、出遊、告白、找房、談分手、出差皆可挑日,也能看出哪些日子適合放空避事。

### 排盤步驟

1. **斗君**：本命盤地支「寅」所在的宮位即本命斗君(見 §1.8)。
2. **流年斗君**：排出流年十二宮後,找與本命斗君「同名」的那一宮(本命斗君是財帛宮,就取流年財帛宮)。
3. **流月**：流年斗君所在即該流年「一月」,順時鐘每格一個月排滿十二個月;當月命宮再排出其餘十一宮,成流月盤。
4. **流日**：流月命宮為當月初一,順時鐘排出初一到三十,每格為一日命宮,再排出其餘宮位,成流日盤。

### 天干選用

流月、流日都用農民曆上的月干、日干(「流」代表外界共同環境,不另起干)。農民曆為陰陽合曆,同一陰曆月可能跨兩個天干,排盤時直接查農民曆即可。各盤各自產生一組四化與祿羊陀填入。

> 為何用流年系統而非小限系統推流月流日:時段越短受外界環境影響越大,由流年推出的流月流日較貼近現象。

### 天地人盤(一次最多疊相鄰三張,時間長的在下)

- 推**流月** → 大限 + 流年 + 流月
- 推**流日** → 流年 + 流月 + 流日

這樣最多三盤、三忌,不會出現五盤五忌的嚇人場面。把三盤的四煞、四化標上去,煞忌最多的宮即最糟,並依煞忌來向判斷哪個領域出事(計算規則見 [02-jiepan.md §2.3](02-jiepan.md))。

### 挑時機(效益時長要對上盤的時長)

| 需求 | 搭配邏輯 |
|------|----------|
| 談生意 / 談判 | 效益至少持續一個月 → 找流月遷移宮或命宮化科;當天要有氣勢與能力 → 找流日命宮或官祿宮化權。可顛倒但效果差;流日官祿宮煞忌匯集就別在那天談 |
| 告白 | 流月命宮、遷移宮或巨門化祿,搭流日夫妻宮化權 |
| 找房 | 流月、流日的子女宮或田宅宮化祿 |
| 找投資 / 股東(進財) | 子女宮、田宅宮(財庫)有化祿或祿存 |
| 談分手 | 挑夫妻宮煞忌多的時候,力道才夠 |
| 出差 | 流月命宮或官祿宮落四馬地(寅申巳亥),或遇天機天梁對拱、太陽太陰對拱,有其一即有機會 |

**原則**：理想組合找不到時退而求其次,挑煞忌較少、相關宮位較乾淨的盤與日子。
