## §2.1 觀盤順序（融合四本書）

按下列順序執行,先粗後細,先扁平後立體。

### 時間軸

1. **本命盤** — 天生個性、價值觀、不可改條件
2. **大限盤** — 十年外在環境影響
3. **流年盤** — 當年外界環境變動
4. 必要時再下到小限 / 流月 / 流日 / 流時
5. **疊宮** — 把多張盤的煞忌與四化同時放到本命盤上一起看

### 元素軸

每個宮位內的星曜,按以下順序拆解:

1. **主星化氣**(星曜本質)
2. **對宮**(內心鏡像) → 詳見 §2.4
3. **雙星組合**(彼此修飾) → 詳見 §2.11、[04-zhuxing.md §4.5](04-zhuxing.md)
4. **四化**(祿、權、科、忌)
5. **煞星**(環境破壞力)

### 立體軸

- **飛化** — A 宮宮干造成 B 宮星曜四化 (A→B,有方向性)
- **自化** — 宮干造成宮**內**星曜四化

→ 詳見 [03-feihua.md](03-feihua.md)

### 快速健康檢查

時間不夠時:直接看流年或小限的三方四正煞忌數量,**煞忌最多的宮位 = 當事人目前最在意的事**。

## §2.2 三方四正 + 對宮 + 夾宮

### 三方四正定義

主宮位 + 順時鐘第四格 + 逆時鐘第四格 (三角形) + 對宮 (箭頭指向)

例:命宮 → 三方 = 命、財帛、官祿;四正 = 加上遷移

**重要規則**:每個宮位都有自己專屬的三方四正,**不是只有命宮才有**。

### 對宮 = 內心 / 外在的鏡像

詳見 §2.4(這是本派的招牌方法論,獨立成節)。

### 夾宮

本書未強調「夾宮」術語,但提到**隔角煞**(命宮在子、夫妻在戌的隔一格現象) — 詳見 §2.9。

### 立體觀

> 「傳統三方四正是『平面』結構;飛化把宮位連結成『立體』結構。」

平面觀解出來的是「現象」,立體觀解出來的是「因果方向」。

## §2.3 煞忌交會計算規則

**四煞 = 擎羊 + 陀羅 + 火星 + 鈴星**,**加化忌** → 共五個影響因子放進三方四正計算。

### 標準規則(按煞忌數量)

| 數量 | 解讀 |
|------|------|
| 0 個 | 平安穩當 |
| 1 個 | 還好,甚至是動力 |
| 2 個(不同宮) | 隊員各有麻煩、有點麻煩 |
| 2 個(同宮) | 除非主星天府 / 天梁,否則該宮算毀 |
| 3 個 | 即使主星是不怕煞的天府也不妙;若集中主宮 + 對宮,這組兩宮都毀 |
| 4 個 | 魔王挑戰宮 |

### 特殊規則 1:火羊鍊大刀

**擎羊 + 火星同宮 + 三方四正有兩個祿 / 權 / 科 → 反而大發**。

火星把擎羊的大刀鍛鍊成為自己所用的好武器。這是煞星反為動力的經典例。

### 特殊規則 2:陀羅的拖延效果

陀羅 = 業力糾纏、不放手。
- 感情上若有陀羅 → 吵架但不一定分手
- 任何宮位有陀羅 → 都有「拖延、不分手」的傾向

### 疊宮計算

| 煞忌 | 疊宮規則 |
|------|---------|
| 擎羊、陀羅、化忌 | 本命、大限、小限、流年各會產生自己的版本,**全部疊上去** |
| 火星、鈴星 | 只有本命盤上有,但透過疊宮會出現在運限位置 |

## §2.4 對宮為明鏡

**這是本派的招牌方法論**。

### 規則

本宮的**內心世界、最深層的價值與期望**,必須看對宮的星曜才能完整解讀。本宮與對宮是一組「外在表現 ↔ 內心世界」的鏡像:本宮是看得見、展現在外的那一面;對宮反映看不見、藏在底層的內心價值與真正期盼。兩者互相牽動、互相影響。

- 本宮的星告訴你「他表現出什麼」;對宮的星告訴你「他骨子裡真正在乎、真正渴望什麼」。
- 兩股力量常一外顯、一節制,互相拉扯,這個張力才是判讀重點。
- 只看本宮會失之扁平;成對讀才能還原內外落差,解出有深度的層次。

> 「本宮的內心世界,所以必須看對宮的星曜,才能做出完整的解釋。例如以貪狼星為例,貪狼為慾望之星,但貪狼對面若是武曲、或者廉貞、抑或是紫微,對於這個貪狼的慾望就會有所不同」

### 應用層面（星曜層面）

| 看哪個宮 | 同時要看 |
|----------|---------|
| 命宮 | 遷移宮 |
| 夫妻宮 | 官祿宮 |
| 七殺單坐 | 對宮一定是天府(紫府 / 廉府 / 武府三種) |
| 貪狼單坐 | 對宮一定是武曲 / 廉貞 / 紫微三選一 |
| 巨門 / 太陰 / 天梁單坐 | 對宮會是天機 / 天同 / 太陽三選一 |

→ 完整對宮配對表見 [04-zhuxing.md §4.3](04-zhuxing.md)

### 十二宮對宮線（宮位層面）

每一宮都有固定的對宮,互為內外鏡像,解盤成對使用:

| 本宮 | 對宮（明鏡） | 鏡像關係 |
|------|------------|---------|
| 命宮 | 遷移 | 外顯個性 ↔ 內心真正想要的 |
| 父母 | 疾厄 | — |
| 財帛 | 福德 | 用錢態度 ↔ 對財富的精神慾望 |
| 子女 | 田宅 | — |
| 官祿 | 夫妻 | 工作的內心狀態 ↔ 感情 |
| 兄弟 | 僕役 | — |

### 為什麼對宮是內心?

因為對宮在三方四正中是「對拱」的位置,結構上互相投射;同時「內外為表裡」是本派的形上學立場。例:
- 官祿宮 = 夫妻宮的內心(感情的內心狀態)
- 遷移宮 = 命宮的內心(個性的內心狀態)

### 抽象範例

若某人外顯位(命宮)坐「重視務實、剛直內斂」的星,而內心位(遷移)坐「重視人際魅力、渴望被眾人喜愛」的星,則他外表看似節制保守,內心其實非常在意人緣與被認同——表裡的張力正是判讀重點。

> 延伸:當本宮為空宮時,就直接借對宮主星過來解,因為兩宮本是同一組表裡的星。詳見 §2.12。

## §2.5 化氣推演 vs 條文死背

### 規則

每顆星都有「**化氣**」一字代表其核心價值。解讀星曜只要扣住化氣不背離,基本上不會錯。**反對條文死背**。

> 「『化氣』在這裡說的也是每個星曜最核心的價值,以此對應各個宮位產生出解釋,所以只要星曜的解釋跟這個中心價值不同,基本上就會是錯的」

> 「利用這樣的方式,我們可以檢視自己的解讀是否有錯,並且可以不需要背誦解釋,因為解釋是被推演出來的,不是套用出來的」

### 推導鏈

```
陰陽五行 + 化氣 + 喜會/忌會 + 宮位本義 + 對宮 + 雙星組合 + 四化 + 煞星
                                  ↓
                                 解讀
```

### 化氣外推範例(書未明示某主星 × 某宮時)

**例**:太陽在田宅宮,怎麼推?
- 太陽化氣 = 貴(光明、領導、照顧人、最亮的星)
- 田宅宮本義 = 家、財庫、居家環境
- 推導:太陽要「光線明朗」「住高地」「漂亮居家」
- 驗證: 實際內容正是如此 ✅

**例**:太陽在僕役宮,怎麼推?
- 太陽化氣 = 貴 + 庇蔭
- 僕役宮本義 = 朋友
- 推導:朋友圈中有名望(化氣為貴 → 體面);照顧朋友(庇蔭)
- 旺位 → 主動領導熱情;落陷 → 心有餘力不足

→ 完整化氣口訣見 [04-zhuxing.md §4.2](04-zhuxing.md)

### 反對死記古書

> 「古人學習紫微斗數的時候,只拿著那兩本小冊子,看來十分淒涼空虛的數十頁去解釋,該如何可以學到出師、精確論斷呢?所以絕對不是靠書上那一點點的資料去記憶、去應用」

古書(《紫微斗數全書》《全集》《斗數宣微》)有許多互相矛盾或脫離邏輯的條文(如「巨、火、羊同宮前齷死」這類嚇人句),應從化氣推演取代死背。

## §2.6 福德宮 > 命宮(進階觀點)

> 【已知差異】前三本書未提此觀點,屬於進階解盤層。

### 規則

高手看的是福德宮,而非命宮。

> 「潛意識影響了我們的命宮態度和真正的價值,這也是為何紫微斗數的高手看的反而是福德宮,而非命宮」

### 處理方式

- **基礎解盤** 仍以命宮為核心(初學者路徑)
- **進階解盤** 引入福德宮 > 命宮觀點:福德 = 靈魂、潛意識、晚年生活、來財方式
- 兩者不衝突:命宮是顯意識的個性,福德是潛意識的驅動;高手看的是後者怎麼推動前者

→ 福德宮 14 主星詳見 [04-zhuxing.md §4.4.11](04-zhuxing.md);宮位涵義見 [05-shiergong.md](05-shiergong.md)

## §2.7 用生命看命盤,不用命盤看生命

> 「學習命理的人要用生命來看待命盤,不要用命盤看生命,請問你看過有人開車只看衛星導航螢幕,不看前方道路的嗎?」

### 三層意涵

1. **命盤是輔助工具,不是絕對真理**
2. **命理師會用自己的價值觀錯估別人** — 「絕大多數的命理師通常是用自己的人生價值去判斷別人的人生價值」
3. **要回到當事人實際的人生去印證** — 對方真正怎麼過、怎麼想、怎麼決定的,才是準度的最終依據

### 宗教信仰會影響準度

有宗教信仰的人會脫離命理學的推演準則(人與環境的變化關係),因為他們依宗教教規行事,不是依個性決定事情。看到對方有強烈宗教信仰時,要降低標準解盤的權重。

### 我宮 vs 他宮術語

**本派不採「我宮 / 他宮」這套南派術語**。改用「A 宮造成 B 宮」的飛化方向描述。skill 應採此用法。

## §2.8 神棍速斷術

「神棍級」的快速解盤路徑,不用排詳細命盤。

### 速斷流程

1. 取當事人的**生年天干**
2. 取**流年的天干** (查農民曆即可)
3. 兩個天干各自起四化 → 取化忌
4. 把流年命宮 = 生肖地支 對照當年地支
5. 結合 → 推算當年運勢

> 「利用流年命盤以及生年的天干跟流年的天干,就可以推算出基本的運勢情況⋯⋯如果你熟練於命盤十二宮位置的人,幾乎可以在心裡就推算出來」

### 為什麼有效?

因為**化忌就是當年最在意的事**(從生年看本性、從流年看當下),只要知道兩個化忌落在哪裡,就抓得到當事人的痛點。

### 反神棍通靈系統

> 「以基督、天主與回教等泛天主教系統來說,人家根本反對這樣的事情,佛教一樣也是反對這樣的事情⋯⋯怎麼可能隨便就會出現九天玄女、關聖帝君這種大神來跟人說話」

速斷術是技術,不是通靈。skill 內部使用時可推算,對外溝通時不要包裝成通靈。

## §2.9 隔角煞

**規則**:命宮與夫妻宮相隔不只一格時 (例如命宮在子、夫妻在戌,隔了一個丑),感情比較不穩定。

不是強調夾宮的術語,傳統論述提到「隔角」概念的場合。實務上常見於命遷線格局。

## §2.10 定性與定心:落星前的兩個前置動作

解盤有固定操作順序:先**定性**(這是哪一張盤)、再**定心**(宮位的基本含義),最後才把星曜性質套進去推。順序顛倒就會淪為死背、到處貼標籤。

### 解盤底層公式

一句話:**解一顆星 = 星曜性質 × 宮位領域**。星性(陰陽、五行、化氣)決定「是什麼樣的力量」,宮位決定「作用在哪個生活面向」。星義不變,落點變,結論才變。整句解讀是推出來的,不是查表查到的。
→ 化氣推演細節見 §2.5。

### 定性:先確定在看哪一張盤

同一個宮位名稱,在不同性質的盤上指的東西不同。落星前必先確認盤的時間屬性(本命 / 大限 / 小限 / 流年,排法見 §2.1)。關鍵分辨:

- 本命盤只代表「天生的態度與價值觀」,不代表事件本身。例:本命夫妻宮講的是感情價值觀與擇偶偏好,不等於配偶本人,也不等於會結幾次婚。
- 運限盤(大限 / 小限 / 流年)才講「現象的發生」,因為現象需要環境與時間觸發。看事件何時成形 → 看運限;看骨子裡是什麼樣 → 看本命。
- 同一現象 = 內因(本命個性) + 外緣(運限環境)。判讀時分開記、再合起來看。

### 定心:先把宮位的基本含義釘死

星曜必須依附宮位才有意義,所以第二步是把每個宮位最素樸的核心含義確認清楚,再讓星曜進來。

- 像蓋房子先打地基:先理解十二宮各管哪個領域,再套星曜,最後才追變化。地基不穩就追細節,整盤越解越歪。
- 宮位含義要回到最基本、最不花俏那一層去定義,避免被後人層層延伸、甚至摻入個人價值判斷的版本污染。
- 本命盤描述現象只能說「有機會、有傾向」,不能斷成「一定會」。把傾向講成必然是初學最常見的錯。

十二宮基本領域速記(本命盤,完整涵義見 [05-shiergong.md](05-shiergong.md)):

| 宮位 | 核心領域 |
|------|---------|
| 命宮 | 自身個性與能力特質,影響全盤 |
| 兄弟 | 對兄弟姊妹的相處態度;母親 |
| 夫妻 | 感情價值觀、擇偶偏好 |
| 子女 | 對子女與對性的態度 |
| 財帛 | 對錢財的態度 |
| 疾厄 | 身體、外型、遺傳 |
| 遷移 | 個人在外的展現、希望外界怎麼看自己、內心世界 |
| 僕役 | 交友態度、各類平輩關係 |
| 官祿 | 學業與工作態度、人生價值的追求 |
| 田宅 | 家世背景、財庫、家人 |
| 福德 | 福氣、精神狀態、靈魂層面 |
| 父母 | 父親、家世與教育環境、長相遺傳 |

## §2.11 雙星同宮:正副主管模型

當一個宮位同時坐兩顆主星,不要硬湊或二選一,用「正副主管」拆解:

- 宮位 = 一個部門,兩顆同宮星 = 正主管 + 副主管。
- 正主管定調主要風格與結果;副主管去調整、修飾、甚至扭轉正主管,可能補強也可能反作用。
- 先後與主從依各組合固定排列決定:先以正主管定主軸,再看副主管把它往哪修。
- 反作用例:一顆天真隨和、與世無爭的星,配上多疑善變的星同宮,原本的安定感反被攪成事事擔憂、缺乏安全感。
- 柔化例:一顆剛直不近人情的星,搭上重人緣、桃花重的星,剛硬被柔化,變得較通人情。
- **格局判定**:某些被看重的星曜組合,若其中一顆只是副主管(非該宮正主管),嚴格講不能算正格,只能說受影響、帶那個味道。判格局要分清正副,別把副位星當主角喊。

→ 雙星組合詳表見 [04-zhuxing.md §4.5](04-zhuxing.md)

## §2.12 空宮借星

宮內沒有任何主星 = **空宮**。主星僅十四顆分到十二宮,本來就會有宮位輪空,空宮不是壞事,毋須害怕。

操作要點:

- 解法 = **借對宮主星過來解**。本宮與對宮本就是同一組互為表裡的星(見 §2.4),借過來後共用一組星義,反而最能呈現內外一致。
- 空宮必與對宮連動著看,不孤立存在。例:本命夫妻宮空宮,其價值觀仍由十二宮連動撐起,**不代表「沒有另一半」**——望文生義嚇自己是常見誤解。
- **例外一:輔星不能當主星借**。即使宮內有文昌、文曲、左輔、右弼這類輔星,仍視為空宮,必須借對宮主星。
- **例外二:四煞落空宮直接當主星**。擎羊、火星、陀羅、鈴星落空宮時,把這四顆煞星特質當主星、直接對應宮位解,不另借星。
- **借星看盤性**:本命盤空宮可借;隨運限才進場的煞星(如運限的擎羊、陀羅)屬原本不存在、後來才進來的力量,不宜當主星硬借,否則會解出虛浮不穩的結果。
