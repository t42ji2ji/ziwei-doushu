## §5.1 十二宮位本義對照表

| 宮位 | 本義 | 衍生義 |
|------|------|
| 命宮 | 個性、主體、長相、能力 | 一輩子的個性、自我要求、掌管十二宮 |
| 父母宮 | 父親(實質父親,不一定生父) | 家世背景、家庭教育、合約(古:官府) |
| 福德宮 | **靈魂、潛意識** | 晚年生活、來財方式(財帛對宮)、心靈品味 |
| 田宅宮 | 家庭、財庫 | 家世、安全感、合夥(官祿兄弟宮)、**方位** |
| 官祿宮 | 工作、人生重心 | 人生價值方向、夫妻對宮(內心)、學業 / 家庭生活 |
| 僕役宮 | 不同性別朋友、所有認識的人 | 部屬、同事、客人 |
| 遷移宮 | 外在表現、外出情況 | **內心世界**、別人對自己的看法、命宮對宮 |
| 疾厄宮 | 身體 | 情緒、長相外型、遺傳、父母對宮、父親遷移宮 |
| 財帛宮 | 金錢 | 用錢態度、福德對宮、理財能力 |
| 子女宮 | 孩子 | 性生活、田宅對宮(內心對家)、財庫、家外面 |
| 夫妻宮 | 感情對象 | **內心對伴侶的態度**、官祿對宮、感情態度 |
| 兄弟宮 | 母親、**同性別**兄弟姊妹 | **現金宮**(財帛對宮、財帛的田宅)、平輩 |

**提醒**:本命盤的宮位都看「態度與價值」,不看具體人物或數量。夫妻宮 ≠ 另一半長相,兄弟宮 ≠ 兄弟姊妹數量(數量看運限)。

## §5.2 本派獨有的延伸涵義

> 這些延伸觀念是本派的識別標誌,在解盤時是常被忽略但實際很有用的維度。

| 衍生義 | 規則 |
|--------|------|
| **官祿 = 生活重心** | 家庭主婦的官祿 = 家務;學生的官祿 = 學業;不只看工作 |
| **兄弟 = 現金宮** | 兄弟宮是財帛宮的田宅(以財帛為命重排) = 手上現金流;母親常掌握現金 |
| **子田線 = 完整財庫** | 子女宮 + 田宅宮 = 古代唯二理財方式(田產與後嗣) |
| **福德 = 靈魂** | 精神與靈魂的宮位;錢財從外而來(來財方式) |  |
| **父母 = 實質父親** | 真正承擔父親權利與義務的人,可能是養父 / 叔 / 舅 |
| **田宅 = 方位** | 三方四正代表方位(前後左右);詳見 [04-zhuxing.md §4.4.10](04-zhuxing.md) |
| **兄弟 = 同性別兄弟 + 母親** | 不同性別兄弟在僕役宮 |
| **子女 = 家外面** | 田宅對宮 = 家外面 = 吃喝享樂 |
| **疾厄 = 父親遷移宮** | 父母(父親)對宮 = 父親在外的情況 |
| **對宮 = 內心** | 所有宮位對宮 = 主要宮位的內心世界(招牌方法論) |
| **遷移 = 內外雙層** | 既是「外在表現」也是「命宮的內心」;遷移好過命宮反而好 |

## §5.3 公司制度比喻

>  **命宮 = 董事長**(統管十二宮)/ **遷移宮 = 總經理**(對外、董事長心腹)/ **財帛宮 = 財務部** / **官祿宮 = 業務部**

**決策由 Group 共同決議,不是單一宮位**。
解盤時不要把命宮當「最重要的唯一指標」,要綜合三方四正一起看。

### 進階拓展(公司隨時間演化)

> 「本命盤像公司各部門的基本組成,大限盤就像是公司每個時空環境內的時間區段內基本組成,運限盤就像是部門間的彼此連結跟影響」

| 層次 | 公司類比 |
|------|
| 本命盤 | 各部門的基本組成(誰是 CEO、誰是 CFO) |
| 大限盤 | 該十年期間的時空版本(公司本季的策略) |
| 流年盤 | 部門間的當期連結(本月的工作協作) |

## §5.4 派生宮位法(以宮為命)

> **把任一宮位當成命宮,重新排出十二宮**。這是衍生義的源頭方法論。

### 推導範例

| 把哪個宮當命宮 | 推導結果 |
|---------------|
| 夫妻宮當命 | 對面成「官祿」(感情的人生重心) |
| 官祿宮當命 | 田宅變「工作上的兄弟」 = 合夥人 |
| 遷移宮當命 | 出門在外的所有人事 |
| 父母宮當命 | 父親的命盤;父親的遷移 = 我的疾厄 |

### 為什麼這個方法有效?

宮位的本義不變(命=個性、財=金錢觀…),但是**主視角**改變了。
從夫妻宮看的「夫妻的人生重心」就是我們本命盤的「官祿」,因為兩個宮在固定的對拱關係中。
這也解釋了「**兄弟 = 現金宮**」是怎麼推來的(從財帛當命排出來,財帛的田宅 = 兄弟)。

## §5.5 六親宮位處理規則

### 六親宮位定義

父母、兄弟、夫妻、子女、僕役(田宅有部分六親屬性)。

### 規則 1:不能機械式套解釋

六親宮位的飛化影響需要視「飛化來源者與命主的**關係深淺與真實接觸**」而定。

> 例:化權在兄弟宮(同性別兄姊或母親),若是尋常人或不熟悉的人,很難直接解釋成他會干涉 / 掌控自己與母親或兄弟的關係。

→ 與飛化的關係限制呼應(見 [03-feihua.md §3.8](03-feihua.md))

### 規則 2:六親宮位的雙面性

六親宮位說的是**我們與這些人彼此的對待方式**;對待對方的方式往往是自己**希望被回饋的方式**。

例:夫妻宮太陽化權 → 自己對伴侶想掌控,同時也希望伴侶是個強勢有主見的人。

### 規則 3:六親宮位分類

| 分類 | 宮位 | 解讀 |
|------|------|
| 出生即有 | 父母、兄弟 | 與生俱來的關係;這兩宮的星曜直接反映本命條件 |
| 出生後產生 | 子女、僕役、夫妻 | 後天形成的關係;四化 = 自己對該宮位產生的價值與對應的態度 |

後三宮的四化更傾向「我們自己選擇與形成的對應方式」,不是命定。
