## §3.1 四化本質(本派招牌)

| 四化 | 本質 | 字源 / 比喻 |
|------|------|
| 化祿 | 緣分付出享受、本來不屬於你而給你 | 春耕。「愛美食而吃東西」 |
| 化權 | 掌控、加倍、希望主導 | 夏耘。「加重」「兩個」 |
| 化科 | 名聲、面子、社會地位、出名 | 秋收。「發生、展現、出現、曝光」 |
| 化忌 | **空缺、需求、動力的源頭** | 冬藏。「飢餓的人去吃東西」 |

### 化忌的核心定義(本派最重要的單一觀念)

> 化忌的「忌」字 = 「己 + 心」 = 「自己的內心」
> 「化忌是內心的空缺,因為空缺會產生力量」
> 「**只有當你失去的時候,才會珍惜跟追求;在難受與不安的同時,卻會帶出更強大的生命力,這才是化忌真正要表達的**」

**核心觀念:化忌不是凶,是空缺即動力**。這是 skill 解盤的最高判準。
解讀化忌時,絕對不可以說「凶」「不好」「破」「殺」這種字眼,要說「在意」「空缺」「動力」「追求」。

### 祿隨忌走

化祿的宮位會去**彌補**化忌宮位的空洞。看到生年化祿,要去找生年化忌的位置,理解祿是為了補哪個缺。

## §3.2 三層四化來源

把四化區分為「三層來源」(不是四化的四種,而是「四化從哪起」的三層)。未明確區分。

| 層次 | 起算依據 | 涵義 |
|------|---------|
| **生年四化** | 出生年天干 | 落在 12 宮,**一生不變**。影響天生個性與在乎之處 |
| **飛化** | 任一宮位的宮干(運限或本命) | A 宮造成 B 宮星曜四化;有方向性 |
| **自化** | 宮位本身的宮干,且該四化星在本宮內 | 宮干造成宮**內**星曜四化 |

飛化的兩個子類:
- **運限飛化**:以運限命宮(大限 / 小限 / 流年)的宮干起四化
- **宮位飛化**:以本命任一宮位的宮干起四化,造成他宮星曜四化

### 時間因素 vs 非時間因素

判讀時可把四化分成兩大類:
- **時間因素**:生年四化(出生天干,與生俱來)、運限四化(大限 / 流年天干,隨時間推進變動)。
- **非時間因素**:飛化、自化。不靠時間自動產生,而是靠宮位之間(或宮位對自己)的天干關係產生。

## §3.3 飛化執行方法

### 公式

```
1. A 宮宮干屬 X(如甲)
2. 查四化表(見 01-paipan.md §1.5):X 干起四化 = X祿/X權/X科/X忌 四顆星
3. 找這四顆星各落在哪個 B 宮
4. → 「A 宮飛祿/權/科/忌入 B 宮」
```

### 範例

- 夫妻宮宮干是甲 → 甲干起四化「廉貞祿、破軍權、武曲科、太陽忌」
- 命宮內有太陽星 → 「**夫妻宮甲干飛太陽忌入命宮**」
- 解讀:感情態度(夫妻宮)造成個性(命宮)的空缺;感情態度常常讓自己內心感到空虛

### 操作優先序

先熟宮位含意與四化本質,飛化即可先跳過細部星曜,做出概略解釋;待熟練後再加入坐守星曜的細節,把事情解釋得更精準。各宮並非孤立,飛化就是把宮與宮之間的「平面連結」與「立體影響」畫出來,看一個部門的狀況如何牽動另一個部門,從而判斷單看盤面表象看不出來的內心波動與真實價值排序。

## §3.4 方向性:A→B ≠ B→A

**這是初學者最易犯錯的地方**。

> 「飛化的重點,就是 A 宮位造成了 B 宮位的變化…官祿宮造成了夫妻宮的變化(如官祿化忌入夫妻),與夫妻宮造成官祿宮的變化(夫妻化忌入官祿),這兩種 A 造成 B 跟 B 造成 A 的狀況會解釋成完全不同,這通常讓初學者搞不清楚」

### 對照範例

| 飛化 | 解讀 |
|------|
| 夫妻化忌入官祿 | 感情態度(對伴侶的關係)讓你在事業上有空缺與動力 → 為情而努力 / 辦公室戀情 |
| 官祿化忌入夫妻 | 事業追求讓你在感情上有空缺 → 工作狂忽略感情 / 為事業犧牲身邊的人 |

**判讀口訣**:「**因為 A 的緣故,造成 B 出現 X 的能量變化**」。先說 A,後說 B。

## §3.5 自化

### 定義

宮位本身的天干(宮干),造成該宮位**內**星曜的四化。

> 條件:宮位內有那顆星 + 宮干又起到那顆星的四化 = 自化。

### 本質:「自以為」、「自己造成、自己引發」

> 「**自化是一種可以隨我們心意調整的力量**」

自化代表「這個宮位本身的價值觀與態度」所引動的結果,是當事人在這個人生面向上自我設定出來的狀態。因為來自內心,所以只要改變心情就能化解 —— 它不是宿命,而是一種可調整的力量。判讀時不只是斷吉凶,更要提醒當事人:問題出在自我認定,調整態度即可鬆綁。

### 自化忌 vs 生年化忌

| 區別點 | 生年化忌 | 自化忌 |
|--------|----------|
| 來源 | 與生俱來 | 宮干自己造成 |
| 性質 | 命定的、別人感受得到 | 自己以為的、別人未必感受得到 |
| 投射 | 直接顯現於該宮,並會實際到對面宮尋求彌補 | 雖也想從對面宮彌補,卻不會把心情真正投射到對面宮去落實 |
| 對問題的態度 | 承認自己有這個缺 | 通常不承認是自己的問題 |
| 強度 | 強 | 「自以為」,可大可小 |

- 對照範例:田宅之宮生年化忌的人,一生覺得家 / 安全感空缺,常特別想透過子女(對面宮)去彌補;同位置若是自化忌,則不會出現這種往對面宮實際彌補的行為。
- 破財同理:自化忌的破財是「自己造成的財庫破洞」,無緣由地破財,但當事人不會承認是自己的問題。

### 自化的時效

- 自化忌的算法與生年四化同理:依本命盤十二宮的宮干推出。
- 呈現在本命盤上的自化 → 通常是終生帶著的特質,不隨運限變動。
- 運限宮干造成的自化 → 只在該運限那段時間內出現,過了就消失。

### 各宮自化忌的常見表現(通則,非個案)

- 自我之宮自化忌:常自我要求高,總覺得自己做得不夠,但其實未必如此;事情容易因此一直被自己推著做。
- 事業之宮自化忌:很重視工作、肯拼、有領導力,但常覺得旁人不夠團結、不夠努力,因而與工作夥伴、上司產生摩擦 —— 而這份落差往往是自己的標準造成的。
- 提醒:以上是「自己的態度」造成的張力,重點在協助當事人看見「是自己覺得」,而非外在真的虧待他。

### 業界三種看法

1. 「自化很嚴重 — 一生就會有問題」
2. 「自化不重要 — 根本不用看」
3. 「自化是『自己覺得』的事情 — 不算太嚴重」

**立場**(採此):
- 自化的本質是「自以為」,可大可小
- 強度差異來自宮內星曜強弱 + 是否同時有煞星
- 加煞星 → 「自以為」會被加重
- 因為是「自己覺得」,外人勸阻不動、講反話也沒用(可怕的一面);但只要當事人改變心情就能化解(可控的一面)

### 自化科 vs 生年化科

| | 解讀 |
|--|
| 自化科 | 自以為外表不錯 |
| 生年化科 | 天生就有不錯的外型 |

### 術語注意

本 skill **未使用「向心自化 / 離心自化」**術語,但有對應概念:
- 自化可投射到對面宮位 ≈ 離心
- 本宮也會出現「自以為」 ≈ 向心

本 skill **不要主動使用向心 / 離心術語**,改用「自以為的投射方向」描述。

## §3.6 飛化解讀公式

### 通用模板

> **「A 宮化 X 入 B 宮 = 因為 A 所代表的人 / 事 / 物的緣故,造成 B 出現 X 的能量變化」**

### X = 哪一化的對應解讀

| 化什麼入 B | B 宮出現什麼 |
|-----------|
| 化祿入 | 緣分、付出、享受 |
| 化權入 | 被掌控、希望主導 |
| 化科入 | 名聲、被看重 |
| **化忌入** | **空缺、需要、需求 → 進而產生追求力量** |

### 化忌入 B 宮的標準解讀

1. 因為 A 所代表的事情 → B 宮出現空缺
2. B 宮 = 你最在意、最需要去填補的標的
3. **「化忌應該是力量的源頭」** — 不是壞事,是動力

### 六親宮位的飛化貴人法

化忌入六親宮位 → 你很在乎那個人 → 在所化忌的宮位中找到貴人星 → 多接近那個人 → 得到改變動力

六親宮位對應:
- 父母 = 父親
- 兄弟 = 母親 / 同性兄姊
- 夫妻 = 對象
- 子女 = 孩子
- 僕役 = 朋友
- 田宅 = 家人

貴人星清單詳見 [06-shaxing-konjie.md §6.4](06-shaxing-konjie.md)。

## §3.7 本命飛化 vs 運限飛化(公司比喻)

> 「本命盤像公司各部門的基本組成,大限盤就像是公司每個時空環境內的時間區段內基本組成,運限盤就像是部門間的彼此連結跟影響」

| 層次 | 涵義 | 起飛宮位 |
|------|------|
| 本命飛化 | 心理素質、與生俱來的價值觀與態度 | 本命任一宮的宮干 |
| 大限飛化 | 那十年期間自己對該事的心情改變 | 大限命宮宮干 |
| 小限飛化 | 那一年期間自己對該事的心情改變 | 小限命宮宮干 |
| 流年飛化 | 當下時空環境的現象與變動 | 流年命宮宮干 |

→ 公司比喻完整版見 [05-shiergong.md §5.3](05-shiergong.md)

## §3.8 飛化的關係限制

別人造成你的飛化,要有「真實接觸」才有效。

> 範例:林志玲讓許多男人怦然心動,但她影響不了大家的感情,因為距離太遠。

**規則**:只看真正能跟自己有接觸與關聯的人。陌生人 / 偶像 / 距離遙遠的人,他們宮位的飛化效應對你來說**接近零**。

| 子女 → 財帛 | 孩子的教育不能等 |
| 田宅 → 財帛 | 家人的需要是賺錢動力 |

### 3.9.F 第八章 — 財帛宮化忌入十二宮(8 cases)

## §3.11 化祿入十二宮語意

### 核心原理

化祿的本質是「增加出來的」:本來不屬於某宮位的能量被額外加注進去。落在哪宮,就是把情感、心力或價值往那宮投注,並希望從中增加緣分與收穫。

判讀務必同時看兩件事:

- **哪一顆星化祿**:四化依星曜屬性變化。同樣化祿,武曲(財星)是「給錢、務實」地付出;太陰是「溫柔、耐心、照顧」;巨門是「好聽話、投其所好」。先認星,才能定義「增加的內容」。
- **哪一層命盤化祿**:本命盤化祿 = 與生俱來的個性與能力使然,是內心自然會去投注的傾向;運限盤化祿 = 在那段時間正在做這件事、正在對該宮投注。本命盤只反映個性與態度,無法直接斷定當下事件;運限盤才對應實際跡象。

### 化祿在六親宮位

六親宮位指父母、兄弟、夫妻、子女、僕役(交友),田宅亦代表家人。化祿落這些宮,代表用自己的個性與能力去增加與該對象的情感緣分,依星曜屬性決定付出方式。

| 落點 | 語意 |
|---|---|
| 化祿入父母宮 | 個性上主動對父親(長輩 / 上司)投注情感,想增加緣分;付出方式看星曜 |
| 化祿入兄弟宮 | 對母親(平輩手足)投注情感與照顧,增加彼此緣分 |
| 化祿入夫妻宮 | 對自己的情感狀態投入心力,是「多情」傾向;本命代表天生重感情,但不等於對當下某位另一半特別好 |
| 化祿入子女宮 | 願意對子女(或合夥、晚輩)付出,投注情感緣分 |
| 化祿入僕役宮 | 個性上對朋友好、願意付出;武曲化祿 = 金錢往來大方,巨門化祿 = 對朋友嘴甜投其所好 |
| 化祿入田宅宮 | 對家庭、家人投注情感(田宅兼財庫意義) |

六親宮的星曜分辨(以付出方式定義):武曲化祿 = 以給錢、實質付出表達;太陰化祿 = 溫柔細膩照顧;巨門化祿 = 好聽的話、投其所好哄人。

僕役宮(交友)特別提醒:
- 本命僕役化祿 = 內心個性對朋友好;運限僕役化祿 = 那段時間正在對朋友付出。
- 若該宮煞忌太多,化祿反而可能助長煞忌力量,未必是好事。
- **破軍化祿落僕役宮要當心**:破軍以「破耗換緣分」運作,容易演成「自己要先有損失(借錢、損耗)才換得到朋友緣分」。

夫妻宮化祿的盤別差異:
- 本命夫妻化祿:對自己的情感狀態投注很多,是多情之人;但對當下另一半好不好是另一回事(可能對象多),若再遇大限夫妻宮不佳,未必對另一半好。
- 大限夫妻化祿:類似本命,但至少這十年對自己的對象不差。
- 流年夫妻化祿:外界環境給了機會去對伴侶表達情感,跡象最明顯。
- 三者中以**武曲化祿**最務實,用金錢展現愛;遇桃花星化祿則感情豐富、難免多情。屬桃花的化祿星有天同、太陰、巨門、破軍、貪狼(廉貞須再遇桃花星才轉桃花,不列入)。

### 化祿在屬於自己的宮位

「屬於自己」的宮位指命宮、官祿宮、財帛宮、遷移宮,以及兼財庫 / 合夥意義的田宅宮、代表精神與財庫的福德宮。化祿落這些宮,代表對自我價值與態度的投注與實現。本命盤代表這份價值天生被產生,運限盤代表當下正在做、正在實現。

| 落點 | 語意 |
|---|---|
| 化祿入命宮 | 對自己很重視;具體展現看哪顆星化祿 |
| 化祿入財帛宮 | 理財能力與價值觀增長(比別人也比自己其他宮強);本命 = 能力與價值觀好,不能直接斷「很會賺錢」;運限 = 透過務實努力而賺到錢 |
| 化祿入官祿宮 | 對工作的態度與能力增長;本命 = 希望在工作上能主導、帶領他人(是期待,不必然有能力);運限 = 工作成就與職場人際變好 |
| 化祿入遷移宮 | 外人對自己的看法、自己想呈現給外界的樣子;運限 = 外出時有機會、有人緣 |
| 化祿入田宅宮 | 對財庫、家業的價值投注(兼財庫意義) |
| 化祿入福德宮 | 運氣福分、精神感官享受、賺錢的方法;依星曜判定品質 |

星曜分辨範例:
- 財帛宮:武曲化祿 = 認真務實的財,理財能力與價值觀都好,但重點是「能力與價值觀」而非直接等於有錢。
- 官祿宮:太陽化祿 = 太陽帶「主導、照顧、帶領」之性,代表希望在工作上主導、照顧帶領大家。
- 遷移宮:武曲(財星)化祿 = 外出有得財機會;桃花星化祿 = 外出有人緣機會。星曜決定外出時增加的是哪種機會。
- 福德宮:偏享樂類星落此化祿可能耽於享受、不願認真奮鬥;天同化祿在福德 = 追求優雅生活,未必是壞事。需依星曜屬性各別判定。

## §3.12 化權入十二宮語意

### 核心語意

- 化權的本質是「加重、掌控」。落哪宮就特別想抓住、主導那宮代表的領域。
- 情感層面化權偏「加重、重視」:重視程度提高,並想用自己的方式去影響、控制。
- 同宮化權的語意隨星曜變化:星曜決定「用什麼方式掌控」,宮位決定「掌控什麼」,兩者合看。
- 判讀順序:先確認宮位主要涵義 → 再看坐守星曜的掌控手法 → 最後加上「加重 / 掌握」的衍生意義。

### 化權在六親宮位(關係的掌控)

化權落六親宮,表示很在意這段關係並想主導對方,但「控制」的態度往往讓對方不舒服,即使出自善意(要求子女照建議走、期待伴侶照自己想法生活),也容易製造緊張與衝突。

| 落點 | 語意 |
|---|---|
| 父母宮化權 | 父親 / 長輩對自己掌控力強;反過來也可解為自我意識重、相當重視自己。本命偏個性,運限偏現象 |
| 夫妻宮化權 | 容易遇較有權勢或個性強的伴侶;自己也想在感情中主導。本命過度想掌控易爭執,運限逢煞忌則感情出問題甚至外遇 |
| 兄弟宮化權 | 多用來看母親(母代父職):母親對自己掌握力強、或自己相當在意該關係 |
| 僕役宮化權 | 容易遇較有權勢、個性強的朋友;與這類人相處張力較大 |
| 命宮化權 | 命宮統管十二宮,化權在命宮影響全盤:對自己要求高、重視人生價值,但相對缺乏傾聽他人的能力 |

判讀要點:
- 出生後才認識的人(朋友、情人)帶「位階」概念,化權會放大彼此的高下與較勁。
- 庇蔭性質的星(太陽、太陰、天梁)落六親宮再化權更需留意:本就帶照顧、保護傾向,化權使「過度想掌控」更明顯,反而傷神、好心做壞事。
- 化權整體不太適合落六親宮 —— 人際一旦過度想掌控就易出問題;需回看星曜本質,而非單看吉凶。
- 取捨建議:化權若躲不開,落父母宮或福德宮殺傷力較小(長輩的控制相對能接受,福德宮屬於自己)。
- 運限夫妻化權又逢煞忌時,把心力從感情轉到工作是化解法:夫妻宮對宮為官祿宮,夫妻的煞忌會牽連工作,專注工作可轉移注意力。

### 化權在自我宮位(掌握 / 決心)

落屬於自己的宮位時,化權轉為正面的「主導、決心、能力放大」,但仍要看是否有足夠能力與運勢支撐。

| 落點 | 語意 |
|---|---|
| 官祿宮化權 | 對工作 / 人生重心有很強的掌控欲與能力,希望按自己規則做事。本命為天生特質,運限對應升官、創業、當主管等實際機會 |
| 財帛宮化權 | 希望掌握金錢、財務上要主導,並放大賺錢能力與機會 |
| 福德宮化權 | 重視自己的精神與心靈價值;同時希望多一份賺錢機會,想掌握好財庫 |
| 田宅宮化權 | 希望擁有房產、或多買一間房存錢;本質是想掌握好財庫 |

星曜與風險:
- 官祿宮:太陽化權偏向願意在工作生活中多照顧人;武曲化權偏向務實、一板一眼。化權帶「雙份」衍生意 —— 可能身兼兩份工作、一人兼兩職。
- 財帛宮:旺位的武曲、太陽化權偏穩定掌握財務;破軍化權偏向利用財務槓桿、誇大理財能力,常以借貸投資。財帛化權逢煞忌過多,易因借貸投資引發財務糾紛。

### 綜合判讀原則

- 自我宮位化權大體偏好,但若命宮不佳、能力或運勢不足以支撐,反而因太重視某方面價值而陷入痛苦,這點初學者最易忽略。
- 同一張盤,先看化權落點是助力(自我宮位、可主導)還是阻力(六親宮位、易控制衝突),再用星曜與煞忌微調結論。
- 一個有利組合:命宮化祿、對宮(遷移宮)化權 —— 內心懂自身價值、外在表現由自己掌控,適合以專業技術發展事業,運限不太差通常有成功機會。

## §3.13 化科入十二宮語意

### 核心定義

化科是四吉之一,但「吉」只代表「現象產生」,不代表「結果一定好」。

- 化科的主軸是「名聲、面子、被看見」—— 它是聚光燈,照到哪宮,那宮的人事物就成為自己想被欣賞、想體面展現的舞台。
- 名聲有好有壞,化科只負責「讓它被注意到」,本身不分善惡;好壞還要看星曜本質與是否遇煞。
- 化科的展現偏「表面層」:好看、賞心悅目,但常缺乏實質支撐(像花一吹就散),不必然帶來具體幫助或財富。
- 與祿、權對照:化祿是「實質得到、源源不絕」,化權是「掌控、被認定能掌握全局」,化科則是「展現給人看、求一個面子與名聲」。
- 判讀順序:先用宮位本義想出「這宮被聚光燈照到會怎樣」,再疊加星曜本質,最後看煞忌修正。

### 化科在我宮(自我相關宮位)

| 落點 | 語意 |
|---|---|
| 命宮 | 重視自我價值的整體展現,對人生各面向都在意「被看見、有體面」;愛面子、講究形象 |
| 遷移宮 | 在意外在表現與「別人怎麼看我」;重視外表、愛打扮、講究門面,希望在外有好名聲 |
| 官祿宮 | 希望工作或學業替自己「掙名聲」;為成就、為被肯定而努力,重視事業 / 課業帶來的體面 |
| 財帛宮 | 把「用錢態度」變成面子;花錢花給人看、講究品味與名牌。同時是一種「把財庫秀出來」的破財傾向 |
| 田宅宮 | 重視居家與出生背景的體面;希望家世、住所看起來不錯、有面子,仍帶破財味道 |

### 化科在六親宮(人的宮位)

重點轉成「這位親人是有名聲 / 有社會地位 / 重面子的人,且他的體面也讓自己有面子」。

| 落點 | 語意 |
|---|---|
| 兄弟宮 | 手足成就不錯、拿得出去;自己也關心並有所期待。六親宮中較理想的落點 |
| 夫妻宮 | 希望配偶有社會地位、有才華或外型出眾,找個「有面子」的對象。風險:重表面而選錯人,條件未必真符合內在期待 |
| 父母宮 | 父母(尤其父親)重面子、有名聲或社會地位;常因此對自己也有高標準成就要求。別誤讀成「家世必然顯赫」 |
| 僕役宮 | 重視人際,想透過交友提升自己的社會地位與名聲;但偏好表面風光的朋友,較易遇只重外表、實質不可靠的損友 |

### 通用推法(套到任何宮位)

把化科視為「聚光燈 + 面子」即可外推:化科入某宮 = 在該宮所主領域,自己想被看見、想體面、在意名聲與外觀展現。
- 六親宮:那位親人是「有名聲 / 愛面子」的人,其體面也替自己加分。
- 我宮:自己在該領域追求形象、名聲與被欣賞。
- 共同弱點:偏表面、易過度修飾,遇煞忌時容易因「愛面子」而做錯決定(為形象衝動投資、亂花錢、選錯人)。

### 本命盤 vs 運限盤的差異

- 本命盤 = 天生態度與價值觀:這個人天生重視什麼面子、追求什麼名聲(例如本命官祿化科 = 天生在乎事業 / 讀書能否考上好學校)。
- 運限盤 = 現象與事件發生:這段期間實際出現的名聲展現(例如運限官祿化科,這段時間容易在工作 / 學業上被看見、得名聲)。
- 財與官的對照:本命財帛化科 = 天生愛花錢買體面(持續習慣);運限財帛化科 = 這段期間出現破財、花錢事件。本命官祿化科偏「能考上、能升的本質」,運限官祿化科偏「實際被看見、得名聲的時機」。
- 運限遷移化科:可能只是個性愛升官、愛被看見,或這段在外名聲不錯,不等於真升官 —— 別把「想被看見」直接讀成「結果達成」。

### 文昌化科的特例

化科落文昌時,語意收斂到「文書、規則、法律、書報媒體、思慮」:
- 命宮文昌化科:重視思慮與守規則、在意學業,會努力展現自己合規、有條理的價值,多半學業不差。
- 反面風險:文昌也代表合約、文書。運限遇文昌化忌或逢煞,可能代表「違反規則、毀約、被開罰單」之類官非 —— 同一顆星,吉化與忌化方向相反,務必分辨。

## §3.14 化忌補強

### 對宮彌補(陰陽表裡)

化忌不能單看本宮,必須用三方四正綜合判讀,其中**本宮與對宮連動最強**,兩者互為陰陽表裡。一個宮位出現化忌(空缺),當事人會把心力、精神投注到**對面宮位**去尋求彌補與支撐;這條彌補路徑會讓兩個對宮互相牽動。

| 本宮(化忌空缺處) | 對宮(心力投注、尋求彌補處) |
|---|---|
| 命宮 | 遷移宮 |
| 夫妻宮 | 官祿宮 |
| 財帛宮 | 福德宮 |
| 田宅宮 | 子女宮 |
| 父母宮 | 疾厄宮 |
| 兄弟宮 | 僕役宮 |

判讀邏輯:先抓「空缺在哪」,再追「心力跑去哪個對宮」。情感宮位出現空缺時,常把精神轉投到對面的事業宮,用工作投入補情感不足;反過來事業宮空缺也會回頭牽動情感。

### 動力的兩種走向(搭配煞星與其他化)

化忌帶來的空缺最終會走向努力、換軌、創業還是內耗,取決於疊加的煞星性質與是否有其他化助力。判讀要「一層一層推」:先定空缺在哪宮(對什麼不滿足)→ 再看疊加煞星是哪種性質(衝動型 / 糾結型)→ 再看有沒有其他化助力(化權想掌控、化祿帶機會)→ 才能推出走向。

- 事業空缺(官祿化忌)+ 衝動型煞星(擎羊、火星、鈴星)+ 命宮或財帛化權:期待長期不被滿足,容易從「想換工作」升級成「乾脆創業」,並可能因此賺到錢。
- 事業空缺 + 糾結型煞星(陀羅):很努力卻又不敢離開,卡在原地反覆折磨、持續硬撐。
- 財星化忌(如武曲化忌):對事業 / 金錢的渴求可能直接推人衝出去開創。
- 財運另一面:財星化祿落命宮、財帛宮、田宅宮等,反而是帶財、有賺錢機會的結構。

### 期待過高 vs 真實不足

空缺的來源可能是能力真的不足、資源真的不夠,或只是期待過高。期待過高時,空缺容易變成自我否定與焦慮;認清空缺本質後,同樣的空缺反而成為動力。同一顆化忌,理解它 = 超越常人的努力與用心;不理解它 = 不安、衝動、被空缺牽著做錯決定。

## §3.15 立太極:六親宮位引申推法

每個宮位都可當成「命宮」來看,再從它順逆排出十二宮,就能引申出各種六親關係。這是六親推算的底層方法:

- 以本命命宮為基準:順時鐘一格是父母宮,逆時鐘一格是兄弟宮。
- 把任一宮位「立太極」當命宮後,它的順時鐘一格即該對象的父母宮,逆時鐘一格即該對象的兄弟宮,其餘依此固定推導。

由此可推出二等親以外的六親宮位(同一宮位常身兼多種角色,要看當下分析的對象來定位,不可死記):

| 對象 | 對應宮位 | 推法 |
|---|---|---|
| 祖父 | 福德宮 | 父母宮的父母宮 |
| 祖母 | 命宮 | 福德宮的夫妻宮 |
| 外祖父 | 命宮 | 兄弟宮的父母宮 |
| 外祖母 | 夫妻宮 | 兄弟宮的兄弟宮 |
| 岳父 | 兄弟宮 | 夫妻宮的父母宮 |
| 岳母 | 子女宮 | 夫妻宮的兄弟宮 |

### 父母宮、兄弟宮的位階之別

- 父母宮 = 父親的命宮,代表父親本人。
- 兄弟宮 = 母親(父親的夫妻宮),代表母親本人。
- 因此判讀父母關係的四化時,要分清落在代表父親的父母宮,還是代表母親的兄弟宮,兩者意義不同。

### 早於自己存在的宮位

命宮以外,父母宮、兄弟宮、田宅宮、福德宮可代表「在我出生前就已存在的人或環境」:
- 父母宮 = 父母。
- 田宅宮 = 小時候的居住環境與家世背景,也可當父母宮的父母宮(祖父輩)。
- 福德宮 = 與出生同時就存在的長輩(祖父)。
