## §6.1 四煞本質

| 煞星 | 本義 |
|------|------|
| 擎羊 | 羊角刀(塞外胡人);豪邁果決的大刀;**衝動破壞** |
| 陀羅 | 業力糾纏(佛教);不甘心不放手的業障;**情感情緒問題** |
| 火星 | 一把大火;熱情而焚身;**燒得快也熄得快** |
| 鈴星 | 小刀慢慢割;精細分解的外科手術刀;**糾結躊躇** |

### 本派觀點:煞星 = 原始情緒

> 「煞星除了空宮,自己一個煞星的時候是直接對宮位產生破壞,宮位出現問題了⋯⋯或許我們可以把煞星當成小孩子看待,就是煞星是一種原始的情緒」

→ 解讀煞星時,不要直接套「凶」,要找它在哪個宮位、表現出哪種原始情緒。

### 擎羊 + 火星 = 火羊鍊大刀(特殊規則 1)

擎羊 + 火星同宮 + 三方四正有兩個祿 / 權 / 科 → **反而大發**。
火星把擎羊的大刀鍛鍊成自己所用的好武器。詳見 [02-jiepan.md §2.3](02-jiepan.md)。

### 陀羅的拖延效果(特殊規則 2)

陀羅 = 業力糾纏、不放手:
- 感情上有陀羅 → 吵架但不一定分手
- 任何宮位有陀羅 → 都有「拖延、不分手」的傾向

### 疊宮計算

| 煞忌 | 疊宮規則 |
|------|
| **擎羊、陀羅、化忌** | 本命、大限、小限、流年各會產生自己的版本,**全部疊上去** |
| **火星、鈴星** | 只有本命盤上有,但透過疊宮也會出現在運限位置 |

完整煞忌數量計算(0/1/2/3/4 規則)詳見 [02-jiepan.md §2.3](02-jiepan.md)。

## §6.2 煞星 × 宮位範例

| 煞星 × 宮位 | 表現 |
|------------|
| 火星在財帛宮 | 亂花錢就跟燒紙錢一樣(快來快去) |
| 鈴星在僕役宮 | 總是算計朋友也受朋友算計(細細糾結) |
| 陀羅在夫妻宮 | 感情總是糾結不清(業力糾纏) |
| 擎羊在官祿宮 | 一不高興就換工作(衝動破壞) |

**推導法**:煞星本義 + 宮位本義 → 推。例如:
- 鈴星(細糾結)在田宅(家)= 為家裡細節煩惱不完
- 火星(快來快去)在子女(孩子 / 性)= 對孩子的耐心容易燃盡 / 性生活突然爆熱又冷掉

## §6.3 空劫

> 【已知差異】這是本派與其他流派最分歧的點之一。其他流派多視空劫為單純的凶星;本派視之為「宮位概念」+「被名字耽誤的吉星」。

**空劫 = 地空 + 地劫**,是一組對星。古書用「天空、地劫」。

### 核心觀念革新

1. **空劫不是「星」,是宮位概念**
2. **空劫的作用 = 讓宮位失去力量**(不是直接破壞)
3. 宮位內的星曜無法發揮正常影響力
4. **比喻**:大樹(星曜)在孤島(被空劫的宮位)上,沒有環境可以發揮乘涼功能

### 空劫是被名字耽誤的吉星

> 「消失力量的吉星(不好)+ 空掉煞星的機會(好)→ 加總起來其實是好朋友」

- 空劫在好宮位 → 把宮位的好力量空掉(壞事)
- 空劫在壞宮位(被煞忌破壞的宮)→ 把壞力量也一起空掉(好事)

→ 看到空劫不要直接判凶。看它落在哪個宮位、那個宮位本來的狀況是好是壞。

### 空劫的三大盲點

破除常見錯誤解讀:

1. **「夫妻宮代表的是感情態度,不是你老公、不是一個人」** — 夫妻宮被空劫不代表「另一半消失 / 沒有對象」,而是對感情的力量空掉
2. **「每個宮位都要看三方四正,一個宮位不產生力量不表示事情不存在」** — 空劫只是把本宮力量空掉,三方四正其他宮的力量還在
3. **「宮位代表的是環境,否則空的是父母宮 → 你媽媽處女懷孕生下你嗎?」** — 父母宮被空劫不代表沒有父母,而是父母對你的影響力空掉

### 空劫四種組合

| 組合 | 規則 |
|------|
| **同宮** | 整個宮位力量被空掉,必須依靠三方四正其他宮位的力量 |
| **對拱**(分別在本宮與對宮) | 本宮與對宮互為表裡,關係最密切;會影響「對宮為明鏡」的內心鏡像功能 |
| **拱**(分別在三方位) | 拱位遇空劫,財帛 / 官祿都受影響,但真正受影響的是命宮 |
| **夾**(在本宮的兩個鄰宮) | 有人祿存在命宮 + 被空劫夾 → **反而好**(把鄰宮的壞力量都空掉,祿存的好留下) |

### 空劫實務應用

「終將一無所有」中「終將」二字 = 時間概念 = 最後會沒有。
**這個情況只會出現在運限盤**(因為「終將」是時間進程)。

→ 本命盤的空劫不是「終將沒有」,是「沒有力量」(性質差異)。

## §6.4 貴人星清單

**七顆貴人星**:天梁、天魁、天鉞、左輔、右弼、天貴、龍德

| 貴人星 | 涵義 |
|--------|
| 天魁 | **陽貴人**,男性,社會地位高 |
| 天鉞 | **陰貴人**,女性,社會地位高 |
| 左輔 | 男性平輩朋友 |
| 右弼 | 女性平輩朋友 |
| 天貴 | 小天梁、無關連的貴人(陌生善意) |
| 龍德 | 有關連且社會地位高的人 |
| 天梁 | 老人星 / 宗教星 / 守護神 |

### 重要規則:**貴人星不能在命宮**

放在命宮表示**自己是大家的貴人,不會有貴人來幫你**。

→ 看到命宮有貴人星,不要說「你有貴人」,要說「你是別人的貴人」。

### 飛化貴人法(交叉應用)

化忌入六親宮位 → 在所化忌的宮位中**找到貴人星** → 多接近那個人 → 得到改變動力。
詳見 [03-feihua.md §3.6](03-feihua.md)。

## §6.5 祿羊陀:等在路上的災難結構

「祿存、擎羊、陀羅」三星位置固定相鄰,以祿存為中心 — **前擎羊、後陀羅**(順時鐘下一格是擎羊,逆時鐘前一格是陀羅)。

- **找法**:用對應天干查出祿存位置,再依「前羊後陀」定出擎羊、陀羅。
- **各盤各有自己的祿羊陀**:本命、大限、小限、流年各自一套,找法與四化相同 — 大限用大限命宮天干、小限用小限命宮天干、流年用農民曆年干。
- 各盤的羊、陀可疊在同一張盤上一起使用,於是命盤上會浮現一般人只看本命盤看不到的**「隱形擎羊、陀羅」**,隨運限偷偷出現。

> 這解釋了為何有些人某段時間突然冒出本命盤上沒有的衝突或拖延 — 那是運限的羊陀疊上來了。四煞性質見 §6.1。

## §6.6 煞忌交會快速避險

命理師最常用、能在幾秒內看出問題的手法:**只用四煞(擎羊、陀羅、火星、鈴星)與化忌的位置,加總一宮三方四正內的煞忌數量,評估該宮健康度**(忌一定要算進去)。完整 0/1/2/3/4 分級規則見 [02-jiepan.md §2.3](02-jiepan.md)。

### 操作流程(找出某段時間最該注意的事)

1. **疊盤** — 看一年用「大限 + 小限」或「大限 + 流年」(小限與流年擇一);看現象用運限盤。
2. 把各盤的四煞與四化(尤其忌)全部標到同一張盤上。
3. 找出**煞忌最多的那一宮** = 災難點。
4. 再看煞忌「**從哪一宮飛來**」 = 哪個相關領域出問題。

### 判讀邏輯(本命 vs 運限)

- 本命盤像「車子本身」的體質問題;小限 / 流年像「開在路上」的當下應變。兩者疊看可推因果 — 本命夫妻宮壞 + 小限財帛宮壞 → 因感情問題導致今年破財。
- 看現象、看事件用運限盤;時段越短受環境影響越大,越短越要用「流」的盤(流年 → 流月 → 流日,見 [01-paipan.md §1.10](01-paipan.md))。

### 避險快判

| 項目 | 判法 |
|------|------|
| **災難類型對應宮位** | 財帛(衝動破財)、田宅/子女(財庫破損存不住)、官祿(工作不順、可能丟工作)、夫妻(感情出問題)、疾厄(生病)、遷移/子女(家外的意外受傷) |
| **斷裂 vs 糾纏** | 會不會「斷」(離職、分手)主要看擎羊、火星;陀羅多半只是磨難、吵架、拖延,不真的斷。三個以上煞忌聚集才高機率破裂,但只要其中有陀羅就會被糾纏拖住而難斷 |
| **凶中藏吉** | 擎羊 + 火星本凶,但三方四正同時遇祿 / 權 / 科其中兩個,反而可能在困境中突破(火羊鍊大刀,見 §6.1);常見於解決問題後升官、離職後創業成功 |
| **血光** | 意外傷災看疾厄 + 遷移/子女的煞忌,再加紅鸞、天喜才會見血;疾厄宮本身沒事時,遷移/子女的煞忌改解為口角、煩惱或破財 |
| **身體部位** | 疾厄宮所在地支對應的身體部位,即可能出狀況之處 |

> 限制:純用煞忌只能判斷「即將出問題」與「問題在哪個領域」,要知道「出什麼問題」的細節仍須回到星曜本身去解。

## §6.7 用運限盤找貴人(實戰)

貴人是人生關鍵助力卻常認不出來(有時討厭的人反而是貴人,逆境也能成就人)。用運限盤可以把貴人找出來。

### 方法一:找貴人星所在的宮位

(七顆貴人星與各自特質見 §6.4)

- 看「**對應需求的宮位**」是否有貴人星:要事業幫手看官祿宮;要有人投資看財帛宮(但田宅宮、子女宮須同時有祿或祿存,否則錢進不了財庫,等於沒人真的投資)。
- **貴人星不能在命宮**(在命宮 = 自己是別人的貴人,見 §6.4)。若整個月運勢太差、不能用命宮,就改看**遷移宮**(在外遇到貴人)。
- 貴人星所在宮位若煞忌太多 → 這貴人幫不上什麼忙。
- **升級用法**:流年找不到就往流月找;要到流日才找,通常是這個月運勢很糟卻又不得不談事或面對感情時。
- **推貴人生肖**:貴人星所在地支對應的生肖,其對宮地支對應的生肖也算(貴人星在午 → 屬馬,或對宮子的屬鼠)。

### 方法二:找祿、科、忌所在的宮位

- **化祿**:緣分深厚,會主動幫你。
- **化科**:為了自己的價值與面子而幫你。
- **化忌**:是空缺、是壓力,會督促你、逼你進步 — 也是一種貴人。
- 取生肖時,本宮地支與對宮地支都可算進去。

### 共同限制

無論用貴人星還是用祿科忌,只要所在宮位煞忌太多,該貴人就不算數(幫忙有限,或要付出其他代價)。
