## §7.1 命運可改的立場

### 核心立場:順天乘風

> 「人該順天而不該是勝天,因為人根本無法勝天,但是為自己找到好的姿態,**順天乘風而起卻是可以的**」

不是「逆天改命」也不是「認命投降」,而是**順著趨勢找最好的姿態**。

### 命盤是地圖與功課表

> 「它最早的設計是讓人可以了解自己,知道自己的人生有什麼該解決的事情。因此,我常說它其實是一張人生的地圖以及功課表」

| 命盤的角色 | 不是 |
|-----------|------|
| 地圖 — 顯示路徑與環境 | 預言 — 預測你會怎樣 |
| 功課表 — 列出該解決的事 | 判決書 — 規定你必須怎樣 |

### 改運是改條件,不是改結果

> 「依照眼前資訊回推之前 + 之後可能發生的事 → 知道事情由『時空環境與人』交錯的條件產生 → 更動條件就好」

**操作流程**:
1. 確認當下處在什麼條件下(本命 + 大限 + 流年的疊宮)
2. 識別哪些條件可以改變(自己的態度、接觸的人、行為選擇)
3. 改變條件 → 結果自然不同

### 本命盤 = 車子,運限盤 = 道路

> 「車子(本命)開在什麼路上會大幅度影響車子的命運」

**早期立場**:**真正的人生地圖是大限盤**。
**後期立場**:「本命 = 心理素質,運限 = 現象」的分工說法。
本 skill 採後者(更平衡)。

> 【已知差異 — 後期演進】這是思想演化的痕跡,後期立場更圓融。

## §7.2 化忌作為改運入口

**這是本派「化忌哲學」如何轉化為實務改運方法的關鍵節**。

> 「**利用紫微斗數命盤找到自己在乎的事情,為了那些事情給予自己面對困境的力量;找出自己跟那個人分不開的情感,為自己找到努力的動力,這是紫微斗數中很重要的改運方式**」

### 改運方法

1. **找出化忌所在的宮位** — 那是自己內心真正的空缺與在乎
2. **理解那個空缺的本質** — 不要逃避,直接看見
3. **利用空缺所產生的動力來改變人生**

### 為什麼這就是改運?

因為:
- 化忌不是凶,是動力源頭
- 知道自己最在意什麼 → 知道該往哪裡用力
- 用力的方向對了 → 結果就會不一樣
- 這就是「改條件 → 改結果」的具體機制

### 命盤再次定位

> 「紫微斗數命盤是一張給我們方向的人生地圖,更是指導我們學習方向的功課表」

→ 這呼應 的「地圖 + 功課表」,但是進一步說明**功課表 = 化忌指出的功課**。

## §7.3 反對江相派恐嚇行銷

### 早期立場

> 「這其實都要怪長年來許多江湖術士喜歡恐嚇行銷,嚇嚇你來做生意⋯⋯不覺得這種說法太無稽之談了嗎?」

### 後期立場

> 「人追逐於簡單快速改運,迷失於對未知的害怕,只希望迅速得到答案,所以慢慢地,紫微斗數就流於所謂論命算命,強調精準度成為了為世人讚嘆的焦點,各類關於紫微斗數的神話被拿來恐嚇民眾,做為命理師取財的工具,**失去了紫微斗數的本意**」

### Skill 應用準則(配合 ETHICS.md)

- ❌ **絕對禁止**:「你這個煞重」「沒救」「血光之災」「短命」「破家敗業」
- ❌ **禁止恐嚇行銷話術**:「再不解就會 XXX」「我幫你化解吧」
- ✅ **正確替換**:「這個位置需要關注」「這是你目前的空缺,也是動力」「值得留意的傾向」
- ✅ **以化忌哲學取代凶煞論**:把「凶」翻譯成「在意 / 空缺 / 動力」

### 反神棍通靈系統

> 「以基督、天主與回教等泛天主教系統來說,人家根本反對這樣的事情,佛教一樣也是反對這樣的事情⋯⋯怎麼可能隨便就會出現九天玄女、關聖帝君這種大神來跟人說話」

紫微斗數是技術,不是通靈。Skill 內部解盤是推算,對外不要包裝成通靈或神諭。

## §7.4 用生命看命盤,不用命盤看生命

> 「學習命理的人要用生命來看待命盤,不要用命盤看生命,請問你看過有人開車只看衛星導航螢幕,不看前方道路的嗎?」

### 三層意涵

1. **命盤是輔助工具,不是絕對真理** — 是導航,不是路況
2. **命理師會用自己的價值觀錯估別人** — 「絕大多數的命理師通常是用自己的人生價值去判斷別人的人生價值」
3. **要回到當事人實際的人生去印證** — 對方真正怎麼過、怎麼想、怎麼決定的,才是準度的最終依據

### Skill 應用準則

- 解盤後**詢問用戶這是否符合他的實際情況**,不是宣告命盤是真理
- 用戶說「不準」時不要強辯,可能是命盤未能反映其當下的真實條件(宗教信仰、特殊環境、有意識的選擇)
- 解盤要留**對話空間**,不要單向講完

### 宗教信仰會影響準度

有宗教信仰的人會脫離命理學的推演準則(人與環境的變化關係),因為他們依宗教教規行事,不是依個性決定事情。

→ 看到對方有強烈宗教信仰時,要降低標準解盤的權重,並告知這個限制。

## §7.5 反對死記古書

> 「古人學習紫微斗數的時候,只拿著那兩本小冊子,看來十分淒涼空虛的數十頁去解釋,該如何可以學到出師、精確論斷呢?所以絕對不是靠書上那一點點的資料去記憶、去應用」

### 古書問題

古書(《紫微斗數全書》《紫微斗數全集》《斗數宣微》)對星曜的解釋很多:
- 互相矛盾
- 脫離邏輯(如「巨、火、羊同宮前齷死」這類嚇人條文)
- 過時(古代社會結構與現代差距大)

### 替代方法:化氣推演

解讀應從「**化氣 + 宮位 + 對宮 + 雙星組合 + 四化 + 煞星**」推演,而不是套條文。
詳見 [02-jiepan.md §2.5](02-jiepan.md) 與 [04-zhuxing.md §4.2](04-zhuxing.md)。

## §7.6 命理普及不藏私的立場

本 skill 的根本立場是**反對藏私 + 反對隱晦**。

### 為什麼重要?

- 紫微斗數長年被「秘傳口訣」「江湖術士只說一半」的方式壟斷
- 這助長了恐嚇行銷:你不懂 → 你害怕 → 我來幫你解
- 公開知識 = 拆解資訊不對稱 = 還給每個人對自己命盤的詮釋權

### Skill 應用準則

- 解盤時**說明推導過程**,不要只給結論
- 用戶問「為什麼這樣解?」 → 把推導鏈攤開(化氣 → 宮位 → 對宮 → …)
- 不要保留「神秘感」 — 越透明越好

## §7.8 給 skill 解盤者的心理檢查清單

每次開始解盤前,內部自檢:

- [ ] 我有沒有在恐嚇用戶?
- [ ] 我有沒有把「化忌」說成「凶」?
- [ ] 我有沒有把自己的價值觀套到用戶身上?
- [ ] 我有沒有留出用戶印證 / 反駁的空間?
- [ ] 我有沒有準備好說「這部分命盤未必反映你的真實情況」?
- [ ] 我有沒有把推導過程攤開,不藏私?
- [ ] 我有沒有遵守 ETHICS.md 的規範?
