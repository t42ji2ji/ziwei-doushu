## §7.1 命運可改的立場

### 核心立場:順天乘風

> 「人該順天而不該是勝天,因為人根本無法勝天,但是為自己找到好的姿態,**順天乘風而起卻是可以的**」

不是「逆天改命」也不是「認命投降」,而是**順著趨勢找最好的姿態**。

### 命盤是地圖與功課表

> 「它最早的設計是讓人可以了解自己,知道自己的人生有什麼該解決的事情。因此,我常說它其實是一張人生的地圖以及功課表」

| 命盤的角色 | 不是 |
|-----------|------|
| 地圖 — 顯示路徑與環境 | 預言 — 預測你會怎樣 |
| 功課表 — 列出該解決的事 | 判決書 — 規定你必須怎樣 |

### 改運是改條件,不是改結果

> 「依照眼前資訊回推之前 + 之後可能發生的事 → 知道事情由『時空環境與人』交錯的條件產生 → 更動條件就好」

**操作流程**:
1. 確認當下處在什麼條件下(本命 + 大限 + 流年的疊宮)
2. 識別哪些條件可以改變(自己的態度、接觸的人、行為選擇)
3. 改變條件 → 結果自然不同

### 本命盤 = 車子,運限盤 = 道路

> 「車子(本命)開在什麼路上會大幅度影響車子的命運」

**早期立場**:**真正的人生地圖是大限盤**。
**後期立場**:「本命 = 心理素質,運限 = 現象」的分工說法。
本 skill 採後者(更平衡)。

> 【已知差異 — 後期演進】這是思想演化的痕跡,後期立場更圓融。

## §7.2 化忌作為改運入口

**這是本派「化忌哲學」如何轉化為實務改運方法的關鍵節**。

> 「**利用紫微斗數命盤找到自己在乎的事情,為了那些事情給予自己面對困境的力量;找出自己跟那個人分不開的情感,為自己找到努力的動力,這是紫微斗數中很重要的改運方式**」

### 前提:化忌是空缺,空缺才會生力量

整套改運邏輯建立在重新理解化祿與化忌之上:

- 化祿 = 內心的滿足與享受。落在某宮,代表那個領域過得舒服、容易得手;但太滿足反而失去前進的動力——吃飽的人不會再去找食物。
- 化忌 = 內心的空缺。落在某宮,代表那一面有缺口、不安、放不下。缺口本身不是壞事,正因為缺,才生出想填滿它的驅動力。
- 關鍵翻轉:**化忌的可用價值往往高於化祿**。真正逼出生命力、推動長期努力的,是那份「永遠覺得還不夠」的不安,而非已到手的安逸。
- 化忌要表達的不是災禍,而是:當你在某處感到失去、難受、不安,才會去珍惜、去追求,帶出比平常更強的能量。

判讀心法:看到化忌,先問「這個空缺指向我內心最在乎、最放不下的是什麼」,而不是先恐慌「這裡要出事」。

### 改運方法(基本步驟)

1. **找出化忌所在的宮位** — 那是自己內心真正的空缺與在乎
2. **理解那個空缺的本質** — 不要逃避,直接看見
3. **利用空缺所產生的動力來改變人生**

本命盤若該處沒有化忌,可改看運限盤、小限、流年;化忌出現的那段時間,就是施力的時機。

### 為什麼這就是改運?

- 化忌不是凶,是動力源頭
- 知道自己最在意什麼 → 知道該往哪裡用力
- 用力的方向對了 → 結果就會不一樣
- 這就是「改條件 → 改結果」的具體機制

### 三大改運主軸

人生最常被在意的三件事,對應三組宮位,也是化忌最值得拿來轉化的方向:

| 主軸 | 對應宮位 | 空缺如何轉成動力 |
| --- | --- | --- |
| 工作 / 事業 | 官祿宮 | 找到自己真心認同(而非別人眼中該做)的工作,化忌的不安會逼出投入與成就 |
| 感情 | 夫妻宮 | 把對感情對象的渴望 / 不安,轉成具體行動與自我經營 |
| 財富與安全感 | 財帛宮、福德宮 | 用理財、花錢目標、心靈寄託去填補空缺,換取前進的方向感 |

### 原理:十二宮連動,對宮自動填補

十二宮彼此牽動,像連通管——一端水位低下去,另一端會流過來補。某宮化忌(出現空缺)時,它的對宮會被帶動過來填補。

- 例:夫妻宮化忌(感情有缺)→ 對宮官祿宮的力量過來填補 → 容易把心力灌注在工作、醉心於事業。
- 實作:核對自己十二宮的四化分布,先找出空缺落在哪一宮,再決定借哪個對宮的力量去補。

### 六親類宮位化忌的改運手法

六親宮(父母、兄弟、夫妻、子女、僕役、田宅)代表人際關係。

- 重新解讀:化忌在六親宮,不該只解成「跟這個人關係不好」。事實常相反——**正因為你在乎這個人,才會在這裡化忌**。空缺 = 重視。
- 手法:在化忌所指的那個人身上找到你的「貴人」。因為彼此有化忌牽連,他的忠言逆耳你才聽得進去,能給你最忠誠的建議、幫你改善問題。別人講一萬次沒用,他講的你會聽——這就是貴人之所以為貴人。

### 非六親類宮位化忌:給空缺一個出口

對非人際的宮位,做法是「替空缺安排一個可努力、可投注的具體目標」:

- 官祿宮化忌:對宮夫妻宮 → 把感情當動力,或讓感情對象成為前進的支點。
- 財帛宮化忌:給自己一個花錢 / 理財目標(添購想要的東西、捐款行善、投入公益或信仰)。目標感帶來最大的前進動力。
- 遷移宮化忌:多往外走、接觸不同人群與生活圈、豐富世界。內在不安會被外在刺激轉成拚命努力。
- 命宮化忌:給自己明確目標與更多信心(行善、寄託信仰皆可),用外在方法把心定下來。
- 福德宮化忌:對宮財帛宮 → 用花錢、理財彌補心靈空虛;並留意它會連帶影響感情態度。

### 飛化:找出「對你有影響力的人」

- 飛化:每個宮位用自己的天干,對其他宮位再產生一次四化。它是平常看不出、卻深深牽動命盤的潛在影響力。
- 改運用途:用飛化找出生活中真正對你有影響力的六親或身邊人,把他們轉化成貴人,借力解決你看不出、放不下、或明知不該做卻做了的問題。
- 範例邏輯:
  - 夫妻宮飛化忌到官祿宮 → 感情的空缺造成工作的空缺,但同一份不安能逼出努力工作的動力。
  - 福德宮飛化忌到官祿宮 → 用內心空缺驅動自己投入工作;看懂內在真正重視什麼,困難就不再可怕。
  - 福德宮飛化忌到田宅宮 → 為了在乎的家人生出無限力量。
- 通則:**人為了真正在乎的人或事,能發出超乎平常的能量。** 找出命盤上「你在乎什麼」,就找到了改運的施力點。

### 自化:自己造成的力量也能反向利用

- 自化:宮位天干使「自己宮內星曜」產生四化,概念是「自己造成」。
- 力量評估:自化本質是一種「自以為」,可大可小。煞星所在宮位的自化較實、較大;虛華星曜組合的自化偏表面、力量較虛。各派輕重看法不一,判讀時要連同宮內星性一起估。
- 反向利用:既然是「自以為」,就拿它做自我暗示——用化忌 / 自化忌定位內心空缺,再主動引爆潛能(像人在危急時會突然爆發平常沒有的力量)。本命沒有就用運限盤。
- 範例:
  - 福德宮自化忌 → 總覺得空虛、影響感情 → 用花錢理財彌補心靈空缺,把空虛轉成行動。
  - 遷移宮自化科 → 多往外接觸新生活圈、累積自信,就不會因福德的空缺而在感情上害怕退縮。

### 自我激勵:改運最後落在自己身上

- 求外力(神明、他人)不如喚醒自己內在的力量;靠自己的決心才能持續。
- 訣竅是「找到真心喜歡 / 在乎的」:像減肥要靠運動,但只有喜歡的運動才做得下去;命盤幫你定位化忌所在的真心在乎之處,再給自己激勵的話語,把潛能逼出來。
- 內在語言是「我可以」「再一下就到了」——靠這種自內心的呼喊撐過長期努力的煎熬,最後往往真能做到。

### 命理師面對命盤的態度(可操作原則)

把命盤當「人生健康檢查表」,而非宿命判決書:

- **空缺 / 煞星 = 待修正項,不是死刑。** 命盤告訴你哪裡需要彌補、哪裡需要修正個性;逐步補上,人生就逐漸圓滿。明白負面能量可以被突破,挫折就不再只是挫折。
- **化忌 = 力量的源頭,不是壞兆頭。** 「忌」字容易被望文生義成傷害,再加情緒衝動才釀禍;但內心的空缺正是動力來源,缺乏才有前進的動機。
- **解決問題靠當事人的決心,命理師只是教練。** 業障像肥肉難消,命理師在旁協助,真正撐下去、消掉它的是當事人自己。
- **拒絕恐嚇式論命。** 拿空宮、煞星組合、宮位化忌等負面元素嚇人、製造對未來的恐慌、再藉人心弱點取財,是要被打破的迷信。
- **對求測者的提醒:** 不要帶著恐慌、用「我是不是註定完蛋」這類問法去問命——那等於主動把弱點遞給對方,容易被騙。命理的真正功能是幫你找出本就具備的力量泉源,給自己面對困境與努力的方向。

底層立場:這套系統源自「如何改善自己」的修身傳統,核心不是預測吉凶,而是用各種方法追求理想、推動自己改變人生。

### 命盤再次定位

> 「紫微斗數命盤是一張給我們方向的人生地圖,更是指導我們學習方向的功課表」

→ 這呼應 §7.1 的「地圖 + 功課表」,並進一步說明**功課表 = 化忌指出的功課**。

## §7.3 反對江相派恐嚇行銷

### 早期立場

> 「這其實都要怪長年來許多江湖術士喜歡恐嚇行銷,嚇嚇你來做生意⋯⋯不覺得這種說法太無稽之談了嗎?」

### 後期立場

> 「人追逐於簡單快速改運,迷失於對未知的害怕,只希望迅速得到答案,所以慢慢地,紫微斗數就流於所謂論命算命,強調精準度成為了為世人讚嘆的焦點,各類關於紫微斗數的神話被拿來恐嚇民眾,做為命理師取財的工具,**失去了紫微斗數的本意**」

### Skill 應用準則(配合 ETHICS.md)

- ❌ **絕對禁止**:「你這個煞重」「沒救」「血光之災」「短命」「破家敗業」
- ❌ **禁止恐嚇行銷話術**:「再不解就會 XXX」「我幫你化解吧」
- ✅ **正確替換**:「這個位置需要關注」「這是你目前的空缺,也是動力」「值得留意的傾向」
- ✅ **以化忌哲學取代凶煞論**:把「凶」翻譯成「在意 / 空缺 / 動力」

### 反神棍通靈系統

> 「以基督、天主與回教等泛天主教系統來說,人家根本反對這樣的事情,佛教一樣也是反對這樣的事情⋯⋯怎麼可能隨便就會出現九天玄女、關聖帝君這種大神來跟人說話」

紫微斗數是技術,不是通靈。Skill 內部解盤是推算,對外不要包裝成通靈或神諭。

## §7.4 用生命看命盤,不用命盤看生命

> 「學習命理的人要用生命來看待命盤,不要用命盤看生命,請問你看過有人開車只看衛星導航螢幕,不看前方道路的嗎?」

### 三層意涵

1. **命盤是輔助工具,不是絕對真理** — 是導航,不是路況
2. **命理師會用自己的價值觀錯估別人** — 「絕大多數的命理師通常是用自己的人生價值去判斷別人的人生價值」
3. **要回到當事人實際的人生去印證** — 對方真正怎麼過、怎麼想、怎麼決定的,才是準度的最終依據

### Skill 應用準則

- 解盤後**詢問用戶這是否符合他的實際情況**,不是宣告命盤是真理
- 用戶說「不準」時不要強辯,可能是命盤未能反映其當下的真實條件(宗教信仰、特殊環境、有意識的選擇)
- 解盤要留**對話空間**,不要單向講完

### 宗教信仰會影響準度

有宗教信仰的人會脫離命理學的推演準則(人與環境的變化關係),因為他們依宗教教規行事,不是依個性決定事情。

→ 看到對方有強烈宗教信仰時,要降低標準解盤的權重,並告知這個限制。

## §7.5 反對死記古書

> 「古人學習紫微斗數的時候,只拿著那兩本小冊子,看來十分淒涼空虛的數十頁去解釋,該如何可以學到出師、精確論斷呢?所以絕對不是靠書上那一點點的資料去記憶、去應用」

### 古書問題

古書(《紫微斗數全書》《紫微斗數全集》《斗數宣微》)對星曜的解釋很多:
- 互相矛盾
- 脫離邏輯(如「巨、火、羊同宮前齷死」這類嚇人條文)
- 過時(古代社會結構與現代差距大)

### 替代方法:化氣推演

解讀應從「**化氣 + 宮位 + 對宮 + 雙星組合 + 四化 + 煞星**」推演,而不是套條文。
詳見 [02-jiepan.md §2.5](02-jiepan.md) 與 [04-zhuxing.md §4.2](04-zhuxing.md)。

## §7.6 命理普及不藏私的立場

本 skill 的根本立場是**反對藏私 + 反對隱晦**。

### 為什麼重要?

- 紫微斗數長年被「秘傳口訣」「江湖術士只說一半」的方式壟斷
- 這助長了恐嚇行銷:你不懂 → 你害怕 → 我來幫你解
- 公開知識 = 拆解資訊不對稱 = 還給每個人對自己命盤的詮釋權

### Skill 應用準則

- 解盤時**說明推導過程**,不要只給結論
- 用戶問「為什麼這樣解?」 → 把推導鏈攤開(化氣 → 宮位 → 對宮 → …)
- 不要保留「神秘感」 — 越透明越好

## §7.8 給 skill 解盤者的心理檢查清單

每次開始解盤前,內部自檢:

- [ ] 我有沒有在恐嚇用戶?
- [ ] 我有沒有把「化忌」說成「凶」?
- [ ] 我有沒有把自己的價值觀套到用戶身上?
- [ ] 我有沒有留出用戶印證 / 反駁的空間?
- [ ] 我有沒有準備好說「這部分命盤未必反映你的真實情況」?
- [ ] 我有沒有把推導過程攤開,不藏私?
- [ ] 我有沒有把空缺 / 煞星說成「待修正項」而非「死刑」?
- [ ] 我有沒有遵守 ETHICS.md 的規範?
