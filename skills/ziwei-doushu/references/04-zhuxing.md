## §4.2 化氣本義口訣(解盤的核心鑰匙)

**解盤順序是主星 → 煞星 → 四化。先確認星本身要什麼。**

| 星 | 化氣口訣 |
|----|
| 紫微 | 皇帝心,化氣為尊;耳根子軟,有皇帝心不一定有皇帝命 |
| 天府 | 王爺 / 地方諸侯 / 財庫星;務實;化煞為用(容納兩顆煞星) |
| 天機 | 善 = 善變 + 善良;邏輯思考、隨時改變方法 |
| 太陽 | 化氣為貴;最亮的星、想當皇帝、庇蔭;旺位喜照不喜坐 |
| 太陰 | 化氣為富;月亮、女性 / 母親 / 家庭 / 聚集財富 |
| 武曲 | 化氣為財;衝鋒陷陣的小將軍、剛毅務實、寡宿星 |
| 天同 | 化氣為福;純真不爭、孩童心、福星、桃花星 |
| 廉貞 | 化氣為囚;外交官、廉潔貞操、磁場強大;亦正亦邪 |
| 貪狼 | 化氣為桃花 + 慾望;解厄星;火貪 / 鈴貪格 |
| 巨門 | 化氣為暗;黑洞 / 吞食星曜光芒;不安全感;需要太陽 |
| 天相 | 化氣為印;宰相、人際協調者、規則守護者;怕煞忌 |
| 天梁 | 化氣為蔭;老人星、宗教星、守護神(貴人星) |
| 七殺 | 化氣為殺 = 魄力 + 堅持;對宮一定是天府 |
| 破軍 | 化氣為耗;大破大立、夢想家、薩滿教破壞之神 |

**用法**:遇到某星在某宮、書中無細目時,**先讀化氣,再扣宮位本義 + 對宮 + 雙星 + 四化 + 煞星推導**。範例見 [02-jiepan.md §2.5](02-jiepan.md)。

## §4.3 對宮為明鏡 — 14 主星解盤方法論

**核心觀念:對宮為明鏡**。單星坐命的解讀**必須先看對宮的星**才完整。

### 應用心法

> 「以貪狼為例,貪狼為慾望之星,但貪狼對面若是武曲、或者廉貞、抑或是紫微,對於這個貪狼的慾望就會有所不同」

例:
- 貪狼對宮武曲 → 務實的慾望 → 想要的是錢與專業
- 貪狼對宮廉貞 → 美感的慾望 → 想要的是品味與魅力
- 貪狼對宮紫微 → 尊貴的慾望 → 想要的是地位與崇拜

→ 完整方法論見 [02-jiepan.md §2.4](02-jiepan.md)

### §4.4.12 父母宮(方法論 + 推演本義)

> 父母宮是**少數本命盤上可以看到現象**的宮位(父母比我們先出生,是與生俱來的)。

**重點觀念**:

- 父母宮代表**父親**(兄弟宮代表母親);父親長相外表可參考命宮、疾厄宮。
- **本派獨特觀點**:父母宮 = 「真正行使父職、實際照顧並影響自己的人」。若父親早逝由他人養大,父母宮可能代表養父、叔叔、舅舅。延伸涵蓋遺傳體質、受教育程度、家世背景,以及社會上的上司,還有人與人間「約定關係」是否被遵守(官非問題)。
- 判讀此宮多採推演,除本宮外要連對宮(疾厄宮)與相關六親宮位合看。

**推演本義**(逢煞忌時下列幾顆星易顯化為「官非或約定被破壞」——不一定打官司,可能是承諾 / 條約被毀):
